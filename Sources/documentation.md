# Documentation

## Introduction
This python project gets new free games from steam daily and sends an email to the user containing the new free games. It doesn't add these games to any steam profile

## Instructions
1. Open the project in VS Code.
2. Review the source files in the `Code/` folder.
3. Run the unit tests to verify functionality.
4. Change the Email in `config.json` to your choice

## Setup
1. Create a Python virtual environment in the project root if not already present.
2. Activate the virtual environment.
3. Install required dependencies.
4. Run main.py in the virtual environment.
